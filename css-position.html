Position property has 4 options

  1. Static
  2. Relative
  3. Absolute
  4. Fixed

Static

Static is the default value for the position property.

Top, bottom, left, and right will NOT work with static.

This element will show up, top to bottom, left to right. Nothing special, and it can't be moved

Static Box

Relative

A relative position is the same as static (top to bottom, left to right), EXCEPT you can move it with top, bottom, left, and right.

IF you move it, it will sit on top of other stuff and the rest of the page will think it's still where "static" would be.

Relative Box

Fixed

A fixed position element will remain relative to the viewport (screen). Even if the user scrolls, the screen doesnt move, so the element doens't move either.

Use top, bottom, left, right to place the element

Unlike relative elements, it does not leave a gap. I.e., the rest of the page (static, relative things) don't know it exists anymore.

fixed Box

Absolute

Absolute position elements work off of the grid system. Just like fixed, they are ignored by everyone. They can have a z-index. They do not leave space/gaps.

The top, bottom, left, and right are determined by the first tag up that has a position set.
